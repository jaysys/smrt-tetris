## 3. 현재 판단 요약
- 현재 확인된 마지막 시험 결과 기준 단계는 `D1`이며 종합 판정은 `Pass`다.
- 다만 `01`, `02` 문서가 최신 설계 변경을 반영해 재수립되었기 때문에, 기존 D1 통과 결과를 최신 케이스 체계에 그대로 1:1 치환할 수는 없다.
- 따라서 현재 범위는 아래 3단계로 관리한다.
  - 이미 수행 완료 및 근거 확보
  - 즉시 수행 가능하지만 최신 기준으로 재판정 필요
  - 개발 단계상 보류

## 4. 현재 기준일 판정
- 기준일: 2026-03-12 00:47 KST
- 현재 단계 해석: `D1 완료 이력 있음`, `D2 이상은 미시험`
- 현재 상태 요약:
  - D0, D1 공식 게이트 통과 이력은 있다.
  - 최신 설계 변경으로 모바일 세로, 접근성 최소 옵션, 이벤트 5종, 광고 보호, 엔진 규칙 정합 기준이 강화되었다.
  - 이 강화 항목 중 일부는 기존 D1 결과서에 직접 증빙이 없으므로 별도 재판정이 필요하다.

## 5. 현재 범위 식별
### 5.1 이미 수행 완료 및 근거 확보
- 아래 항목은 기존 시험 결과서 기준으로 수행 완료 근거가 있다.
- `TC-001` 랜딩 진입
- `TC-002` 게스트 즉시 플레이
- `TC-003` 튜토리얼 스킵
- `TC-004` 좌우 이동
- `TC-005` 회전 입력
- `TC-006` 소프트 드롭
- `TC-007` 하드 드롭
- `TC-008` Hold 기능
- `TC-009` Ghost Piece
- `TC-010` 라인 제거
- `TC-012` 게임 오버
- `TC-017` 재도전 흐름
- 명령/게이트:
  - `pnpm test`
  - `pnpm db:reset`
  - `pnpm db:migrate`
  - `pnpm db:seed`
  - `pnpm build`
  - `pnpm test:e2e`

해석 주의:
- 위 항목은 당시 시험 기준으로 Pass 근거가 있는 항목이다.
- 최신 설계 강화 기준이 추가된 케이스는 아래 `재판정 필요`로 분리한다.

### 5.2 즉시 수행 가능하지만 최신 기준으로 재판정 필요
- 아래 항목은 현재 단계와 구현 범위를 고려하면 시험은 가능하지만, 최신 설계 강화 기준으로 다시 확인해야 한다.
- `TC-011` 점수 계산
  - 이유: 최신 설계는 점수 항목 일부가 아니라 판정 순서 전체 정합을 요구한다.
- `TC-016` 결과 화면 CTA
  - 이유: 첫 CTA 영역 보호와 결과 카드 구조를 최신 기준으로 다시 확인해야 한다.
- `TC-017A` 첫 3판 최고 기록 강조 연출
  - 이유: 최신 설계에서 새로 강화된 독립 게이트다.
- `TC-026` 설정 저장
  - 이유: 접근성/설정 최소 옵션이 조기 게이트로 상향되었다.
- `TC-027` 고대비 모드
  - 이유: 접근성 최소 옵션의 현재 구현 여부를 다시 확인해야 한다.
- `TC-028` 모바일 세로 대응
  - 이유: `390x844`는 이제 D1 품질 게이트 성격을 가진다.
- `TC-028A` 모바일 세로 대체 뷰포트
  - 이유: `360x800` 케이스가 독립 게이트로 추가되었다.
- `TC-034` 분석 이벤트 수집
  - 이유: 이벤트 5종은 최신 설계에서 필수 품질 게이트다.
- `TC-036` 광고 금지 구역 보호
  - 이유: 플레이/첫 결과 화면/재도전 직전 보호 규칙이 최신 설계에 추가되었다.

| 최신 계획/케이스 기준 재시험 준비 | 가능 | `01`, `02` 최신본 기준 |

| 광고 금지 구역 재판정 | 가능 여부 미확정 | 광고/프로모션 UI 존재 전제 필요 |

## 7. 현재 시점의 권장 시험 순서
1. `TC-028`, `TC-028A`, `TC-027`, `TC-026`
2. `TC-011`, `TC-016`, `TC-017A`
3. `TC-034`
4. `TC-036`

이 순서를 권장하는 이유:
- 최신 설계에서 D1~D3 조기 게이트로 승격된 항목을 먼저 닫아야 한다.
- 기존 D1 통과 이력과 최신 설계 간 차이를 가장 빠르게 줄일 수 있다.

## 8. 결론
- 현재 시점에서 테스터가 즉시 수행 가능한 시험은 있다.
- 다만 최신 설계 변경으로 인해 기존 D1 Pass 이력이 모든 최신 케이스를 자동으로 닫아주지는 않는다.
- 현재 운영상 가장 중요한 문서는 `TC-011`, `TC-017A`, `TC-026 ~ TC-028A`, `TC-034`, `TC-036`의 재판정 필요 상태를 관리하는 것이다.

# assets-source

- 본 폴더는 외부 시각 에셋 원본과 라이선스 기록을 보관하기 위한 위치다.
- 실제 SVG, PNG, 오디오를 반입할 때는 아래 규칙을 따른다.

## 규칙
- 원본 파일명은 공급 소스와 추적 가능한 형태로 유지한다.
- 파일을 추가하기 전에 `LICENSE_INVENTORY.md`에 먼저 기록한다.
- CC BY 계열은 출처 URL, 저작자 표기 문구, 공개 credits 반영 여부를 함께 기록한다.
- 라이선스가 불명확한 자산은 이 폴더에 저장하지 않는다.

## 현재 상태
- 2026-03-12 기준 실제 외부 자산 반입 없음
- 컬러 토큰, CTA 스타일, credits 구조만 선행 고정 완료

# 개발자전달 게임폰트 UI스킨 적용지시서

## 1. 목적
- 본 문서는 화면 전체를 `기본 웹앱`이 아니라 `게임 서비스`처럼 보이게 만들기 위해, 전역 폰트와 UI 스킨 소스를 개발자에게 고정 전달하는 문서다.
- 목적은 시스템 기본 폰트와 단순 CSS 박스를 버리고, 공개 라이선스 기준의 게임형 폰트와 프레임 에셋을 실제 화면에 적용하게 하는 것이다.

## 2. 핵심 지시
- 전역 폰트는 시스템 기본 폰트를 사용하지 않는다.
- 게임 화면, 결과 화면, 랭킹 화면에 보이는 박스와 버튼은 기본 CSS 사각형 대신 `Kenney UI Pack` 계열 프레임을 적용한다.
- 공식 게임 사이트의 실제 UI 이미지나 스크린샷을 잘라 쓰지 않고, 추천한 공개 에셋 소스만 사용한다.

## 3. 전역 폰트 기준
### 3.1 기본 폰트 조합
- Display / 타이틀 / CTA: `Black Han Sans`
- HUD 숫자 / 점수 / 랭킹 숫자 / 플레이 시간: `Orbitron`
- 본문 / 짧은 설명 / 키보드 안내 / 보조 라벨: `Signika`

### 3.2 적용 규칙
- 랜딩 Hero, 모드명, 결과 Hero, 주요 CTA는 `Black Han Sans`를 사용한다.
- 점수, 라인, 레벨, 플레이 시간, 랭킹 숫자는 `Orbitron`을 사용한다.
- 키보드 안내, 짧은 문장, 설정 설명은 `Signika`를 사용한다.
- 폰트 로딩 실패 시에만 일반 sans-serif를 fallback으로 사용한다.

## 4. 폰트 소스
- Black Han Sans
  - https://github.com/naver/black-han-sans
- Orbitron
  - https://github.com/googlefonts/orbitron-vf
- Signika
  - https://github.com/googlefonts/signika

## 5. 게임 화면 UI 스킨 기준
### 5.1 반드시 스킨 적용할 요소
- 상단 HUD 박스
- 키보드 안내 미니 박스
- 일시정지 버튼
- 일시정지 오버레이 카드
- 결과 Hero 카드
- 결과 CTA 버튼
- 랭킹 Top 3 카드
- 랭킹 리스트의 내 기록 강조 카드

### 5.2 1순위 스킨 소스
- `Kenney UI Pack`
  - https://kenney.nl/assets/ui-pack
- `Kenney UI Pack - Sci-Fi`
  - https://kenney.nl/assets/ui-pack-sci-fi
- `Kenney UI Pack - Adventure`
  - https://kenney.nl/assets/ui-pack-adventure

## 6. 적용 방식
### 6.1 게임 화면
- 상단 HUD 박스와 키보드 안내 박스는 `Kenney UI Pack - Sci-Fi` 프레임을 우선 적용한다.
- 일시정지 버튼은 같은 계열 프레임을 축소 적용해 화면 안에서 떠 보이지 않게 맞춘다.
- Next 1 미니 프리뷰는 프레임보다 블록 모양 자체가 더 눈에 띄어야 하므로, 매우 얇은 보조 프레임만 허용한다.

### 6.2 결과 화면
- 결과 Hero와 `재도전` 버튼은 `Kenney UI Pack` 또는 `Adventure` 계열 프레임을 써서 보상 화면처럼 보이게 만든다.
- `재도전`은 가장 큰 버튼 프레임, 나머지 CTA는 한 단계 약한 프레임을 쓴다.

### 6.3 랭킹 화면
- Top 3는 메달/트로피/월계수 오브젝트와 함께 `Adventure` 또는 `Sci-Fi` 계열 프레임으로 시상대처럼 보여야 한다.
- 일반 리스트는 더 얇은 프레임을 쓰고, 내 기록 카드만 한 단계 강한 프레임으로 구분한다.

## 7. 금지사항
- 기본 CSS `border`, `box-shadow`, `border-radius`만으로 박스와 버튼을 끝내는 방식
- 시스템 기본 폰트, Inter, Arial, sans-serif 기본 화면 유지
- 공식 Tetris, Candy Crush, Royal Match, Subway Surfers의 UI 이미지 직접 사용
- 한 화면 안에서 서로 다른 스킨 팩을 무질서하게 섞는 방식

## 8. 개발자 액션 요청
1. 전역 폰트를 `Black Han Sans`, `Orbitron`, `Signika` 체계로 교체한다.
2. 게임 화면의 HUD 박스, 키 안내 박스, 일시정지 버튼에 `Kenney UI Pack - Sci-Fi` 계열 프레임을 적용한다.
3. 결과 화면과 랭킹 화면 카드/버튼도 `Kenney` 프레임으로 바꾼다.
4. 기본 CSS 사각 박스로 남아 있는 버튼과 박스를 모두 교체한다.

## 9. 최종 전달 문안
- 전역 폰트는 시스템 기본 서체를 쓰지 말고 `Black Han Sans`, `Orbitron`, `Signika` 조합으로 교체해 달라. 게임 화면, 결과 화면, 랭킹 화면에 보이는 박스와 버튼은 기본 CSS 박스 대신 `Kenney UI Pack`, `Kenney UI Pack - Sci-Fi`, `Kenney UI Pack - Adventure` 프레임을 사용해 달라. 공식 게임 사이트의 UI 이미지를 직접 쓰는 것이 아니라, 공개 라이선스가 확인된 추천 소스의 폰트와 프레임만 사용해 전체 화면을 게임 서비스 톤으로 맞춰 달라.

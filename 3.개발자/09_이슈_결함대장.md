# 이슈/결함대장

## 1. 목적
본 문서는 개발 및 테스트 과정에서 발생하는 blocker, 버그, 성능 이슈, 문서 불일치 이슈를 추적 관리한다.

## 2. 운영 원칙
- 발견 즉시 등록한다.
- 재현 절차 없는 이슈는 `조사중`으로 분류한다.
- 해결 시 원인, 수정 범위, 검증 결과를 반드시 남긴다.

## 3. 우선순위 기준
| 등급 | 의미 |
|---|---|
| `Blocker` | 개발 진행 또는 테스트 진행이 불가능한 상태 |
| `Critical` | 핵심 기능 불가, 데이터 오염, 랭킹 오염 |
| `Major` | 주요 기능 이상, 우회 가능하나 릴리스 영향 큼 |
| `Minor` | UI, 문구, 비핵심 동작 문제 |
